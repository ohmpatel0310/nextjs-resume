# Next.js Resume – Ohm Patel

## Overview
This project is a personal resume website built using Next.js and Tailwind CSS. It recreates my professional resume in a clean, responsive, and accessible web format.

## Features
- Semantic HTML structure (header, main, section, footer)
- Fully styled using Tailwind CSS utility classes
- Responsive and clean layout
- Sections include Education, Experience, Projects, Leadership, and Skills
- Clickable contact links (email, LinkedIn, GitHub)

## Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/ohmpatel0310/nextjs-resume.git

2. Navigate into the folder:
   cd nextjs-resume

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open http://localhost:3000

## Known Issues
- Optimized primarily for desktop view (mobile responsiveness can be improved)
